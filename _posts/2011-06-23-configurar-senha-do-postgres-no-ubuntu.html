Quando você instala o postgresql 8.6 no Ubuntu ele acaba não pedindo a senha inicial quando você tenta logar ocorre um erro

An error has occurred:

Error connecting to the server: FATAL: password authentication failed for user "postgres"
FATAL: password authentication failed for user "postgres"

Para resolver isso é bastante simples, vamos configurar nosso postgres com uma nova senha

 

Logue no postgresql usando a conta root:
sudo -u postgres psql template1
Depois quando estiver no psql digite o seguinte comando SQL:
ALTER USER postgres WITH PASSWORD 'nova_senha';
onde 'nova_senha' é a sua nova senha.

por exemplo
ALTER USER postgres WITH PASSWORD 'postgres';
Saia do console do postgresql digitando \q

Prontinho postgres está pronto para ser utilizado.
