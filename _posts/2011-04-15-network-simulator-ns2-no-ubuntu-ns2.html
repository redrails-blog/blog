Instalação:

I) Devemos abrir o Synaptic para adicionar os novos repositórios.

II) Para Adicionar um repositório siga os passos: Settings->Repositories



III) Na janela subsequente realize o seguinte procedimento.
1) Vá para a aba Other Softwares
2) Botão add
3) adicione (um de cada vez) os seguintes dados no campo APT Line

deb http://ppa.launchpad.net/wouterh/ppa/ubuntu lucid main
deb-src http://ppa.launchpad.net/wouterh/ppa/ubuntu lucid main

IV) Agora vamos instalar os pacotes.

1) Busque pelo Package ns
2) Clique com botão diretor o quadro branco ao lado do nome ns na lista de pacotes.
3) Selecione a opção Mark for installation
(Repita os passos 1,2 e 3 mas busque pelo pacote nam )

4) Clique em Apply

Mais detalhes em : http://launchpad.net/~wouterh/+archive/ppa

Testando:

Para testar se tudo está funcionando bem faça o download do arquivo ns-sample.tcl e execute o seguinte comando no diretório onde o arquivo foi salvo.

ns ns-sample.tcl
