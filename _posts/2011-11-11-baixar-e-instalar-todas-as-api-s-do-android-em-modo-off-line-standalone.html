O Gerenciador de Download do SDK Manager é bem limitado, além de alcançar uma velocidade inferior que um bom gerenciador de download, ele trava com facilidade e ai todo seu esforço é perdido. Uma opção é baixar as versões standalone das API e instala-las manualmente. Primeiro baixe a versão(sões) que necessita.

Depois de baixado renomeie a pasta descompactada de acordo com sua API level.
EX:

  • Baixarei a Android SDK Platform 3.2_r1
  • O arquivo baixado é android-3.1_r01-linux.zip (não se importe com o linux, ele é compatível com Windows e MAC tbm)
  • descompacte o .zip na pasta plataform/ no diretório onde está seu SDK.
  • renomeie a pasta para android-13
