Bom vira e mexe temos que realizar algum tipo de manipulação de String. Não vou postar as melhores práticas em relação a esse contexo, apenas algumas soluções que encontrei por ai em foruns,grupos e blogs. espero que seja de grande valia para você!

Primeiramente vou passar um pouco do básico de String.

Uma String detém e manipula uma seqüência arbitrária de bytes,
normalmente representando caracteres.

EX:
[source lang="ruby"]
str = String.new("ascatezuma")
str = "ascatezuma"
"ascatezuma"
[/source]

Devido a questões nomeclatura, os usuários devem estar cientes de
alguns dos métodos podem modificar o conteúdo de uma String.
Normalmente,métodos terminados em "!" (Exclamação),  podem modificar
seu receptor, enquanto que aqueles sem o "!" , normalmente retorna
uma nova String.

EX:

[source lang="ruby"]
a = "askatezuma"
b = "buchada"

b = a.reverse
puts a #=> "askatezuma"
puts b #=> "amuzetaksa"

puts a.reverse! #=> "amuzetaksa"
[/source]

Isso é muito útil imagine que você deve pegar uma string, essa string e
todos seus caractéres devem estar em caixa alta (upcase) normamente eu faria
assim:
[source lang="ruby"]
a = "askatezuma"

a = a.upcase
=> "ASKATEZUMA"
[/source]
Podemos observar que repitimos o "a" duas vezes, o Ruby nos ajuda a deixar isso um pouco mais DRY (sem repetições) assim:

[source lang="ruby"]
a.upcase!
=> "ASKATEZUMA"
[/source]
